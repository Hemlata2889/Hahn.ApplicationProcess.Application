This solution has 2 web projects.
1. Hahn web application which is built using javascript and bootstrap html.
	This project has connectivity to in memory database using Entityframework.
	Swagger is implemented.
	All the validations are covered.
	Serilog is implemented.
To run this solution : set this project as startup project and F5.
Swagger UI : All the crud operations can be tested through this.

------------------------------------------------------------------------------
2. Aurelia template

This project has only the form and validation is implementd.
Couldn't connect to database using aurelia framework.

To run this project go to command prompt --> change directory of this project "aurelia-app"
hit command "au run --open"
